
# Tetris
Tetris game with leaderboards
## Installation:
Clone this repository to your desired directory

```git clone https://github.com/KalleHahl/tiralabra```

Install dependencies

```poetry install```

Start program

```cd TETRIS```

```poetry run invoke start```
